# Mobile lab 1

## Описание

Это приложение - набор небольших мини апп:

- квиз;
- калькулятор;
- игра "угадай число".

Проект был создано с помощью [Expo](https://expo.dev) [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Инстракция по настройке

1. Установка зависимостей

   ```bash
   npm install
   ```

2. Запуск приложения

   ```bash
    npx expo start
   ```
